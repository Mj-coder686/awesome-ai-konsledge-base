## 8. 风险与应对

| 风险 | 概率 | 影响 | 应对策略 |
|------|:---:|:---:|---------|
| Ollama 本地模型效果差 | 中 | 中 | 同时支持 OpenAI API 作为备选 |
| Spring AI API 变动 | 低 | 高 | 锁定版本，参考官方文档 |
| 向量检索精度不足 | 中 | 中 | 调整 chunk size / overlap，后续加 Rerank |
| Tika 解析异常 | 低 | 低 | 做好异常兜底，记录错误日志 |
| 时间不够 10 周 | 中 | 中 | 前端可简化（优先后端），Phase 5 前端可后移 |
