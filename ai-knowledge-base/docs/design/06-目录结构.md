## 5. DTO / VO / PO 命名规范

| 类型 | 后缀 | 说明 | 示例 |
|------|------|------|------|
| **PO** (Persistent Object) | 无特殊后缀 | 数据库实体，对应一张表 | `User.java`, `Document.java` |
| **DTO** (Data Transfer Object) | `DTO` | 请求入参 | `LoginDTO.java`, `ChatSendDTO.java` |
| **VO** (View Object) | `VO` | 响应出参 | `UserVO.java`, `MessageVO.java` |
| **Query** | `Query` | 复杂查询参数(分页+条件) | `KnowledgeQuery.java` |

### 所在位置

```
每个模块的 domain/ 目录下:
domain/
├── po/      ← PO 实体
├── dto/     ← DTO 请求参数
└── vo/      ← VO 响应结果
```
