### Step 1.5 创建 MyBatis Plus 配置

**文件：** `src/main/java/com/mj/aiknowledgebase/config/MyBatisPlusConfig.java`

**知识点：**
- `@Configuration` — 声明这是一个配置类，Spring 启动时会自动扫描
- `MybatisPlusInterceptor` — MyBatis Plus 的拦截器容器
- `PaginationInnerInterceptor` — 分页插件，之后用 `Page<T>` 做分页查询时需要它
- `MetaObjectHandler` — 自动填充处理器，让 create_time 和 update_time 自动写入

```java
package com.mj.aiknowledgebase.config;

import com.baomidou.mybatisplus.annotation.DbType;
import com.baomidou.mybatisplus.core.handlers.MetaObjectHandler;
import com.baomidou.mybatisplus.extension.plugins.MybatisPlusInterceptor;
import com.baomidou.mybatisplus.extension.plugins.inner.PaginationInnerInterceptor;
import org.apache.ibatis.reflection.MetaObject;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import java.time.LocalDateTime;

@Configuration
public class MyBatisPlusConfig {

    // 分页插件
    @Bean
    public MybatisPlusInterceptor mybatisPlusInterceptor() {
        MybatisPlusInterceptor interceptor = new MybatisPlusInterceptor();
        interceptor.addInnerInterceptor(new PaginationInnerInterceptor(DbType.MYSQL));
        return interceptor;
    }

    // 自动填充 create_time / update_time
    @Bean
    public MetaObjectHandler metaObjectHandler() {
        return new MetaObjectHandler() {
            @Override
            public void insertFill(MetaObject metaObject) {
                this.strictInsertFill(metaObject, "createTime", LocalDateTime.class, LocalDateTime.now());
                this.strictInsertFill(metaObject, "updateTime", LocalDateTime.class, LocalDateTime.now());
            }

            @Override
            public void updateFill(MetaObject metaObject) {
                this.strictUpdateFill(metaObject, "updateTime", LocalDateTime.class, LocalDateTime.now());
            }
        };
    }
}
```

### Step 2.3 创建 JWT 工具类

**文件：** `src/main/java/com/mj/aiknowledgebase/util/JwtUtil.java`

**知识点：**
- JWT（JSON Web Token）= Header.Payload.Signature 三段拼接的字符串
- `Jwts.builder()` — 构建 JWT
- `setSubject()` — 存入用户标识
- `setExpiration()` — 设置过期时间
- `signWith()` — 用密钥签名，防止篡改
- `parseClaimsJws()` — 解析并验签

```java
package com.mj.aiknowledgebase.util;

import io.jsonwebtoken.Claims;
import io.jsonwebtoken.Jwts;
import io.jsonwebtoken.security.Keys;
import org.springframework.beans.factory.annotation.Value;
import org.springframework.stereotype.Component;

import javax.crypto.SecretKey;
import java.nio.charset.StandardCharsets;
import java.util.Date;

@Component
public class JwtUtil {

    @Value("${jwt.secret}")
    private String secret;

    @Value("${jwt.expiration}")
    private Long expiration;

    // 获取签名密钥
    private SecretKey getSigningKey() {
        return Keys.hmacShaKeyFor(secret.getBytes(StandardCharsets.UTF_8));
    }

    // 生成 Token
    public String generateToken(Long userId, String username) {
        return Jwts.builder()
                .subject(userId + ":" + username)    // 存入 userId 和 username
                .issuedAt(new Date())                 // 签发时间
                .expiration(new Date(System.currentTimeMillis() + expiration))  // 过期时间
                .signWith(getSigningKey())            // 签名
                .compact();
    }

    // 解析 Token，返回 Claims（载荷）
    public Claims parseToken(String token) {
        return Jwts.parser()
                .verifyWith(getSigningKey())
                .build()
                .parseSignedClaims(token)
                .getPayload();
    }

    // 从 Token 中获取 userId
    public Long getUserId(String token) {
        String subject = parseToken(token).getSubject();
        return Long.parseLong(subject.split(":")[0]);
    }

    // 从 Token 中获取 username
    public String getUsername(String token) {
        String subject = parseToken(token).getSubject();
        return subject.split(":")[1];
    }

    // 验证 Token 是否过期
    public boolean isTokenExpired(String token) {
        return parseToken(token).getExpiration().before(new Date());
    }
}
```

### Step 2.4 创建 JWT 拦截器

**文件：** `src/main/java/com/mj/aiknowledgebase/config/JwtFilter.java`

**知识点：**
- `OncePerRequestFilter` — Spring 提供的 Filter 基类，保证每次请求只执行一次
- `doFilterInternal` — 每个请求都会经过这里
- `request.setAttribute()` — 把解析出来的 userId 存入 request，后续 Controller 可以取
- `doFilter()` — 放行，让请求继续往后走

```java
package com.mj.aiknowledgebase.config;

import com.mj.aiknowledgebase.util.JwtUtil;
import jakarta.servlet.FilterChain;
import jakarta.servlet.ServletException;
import jakarta.servlet.http.HttpServletRequest;
import jakarta.servlet.http.HttpServletResponse;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Component;
import org.springframework.web.filter.OncePerRequestFilter;

import java.io.IOException;

@Component
public class JwtFilter extends OncePerRequestFilter {

    @Autowired
    private JwtUtil jwtUtil;

    // 不需要认证的路径
    private static final String[] WHITE_LIST = {"/api/register", "/api/login"};

    @Override
    protected void doFilterInternal(HttpServletRequest request,
                                    HttpServletResponse response,
                                    FilterChain filterChain) throws ServletException, IOException {

        String path = request.getRequestURI();

        // 白名单直接放行
        for (String whitePath : WHITE_LIST) {
            if (path.equals(whitePath)) {
                filterChain.doFilter(request, response);
                return;
            }
        }

        // 从 Header 取 Token
        String authHeader = request.getHeader("Authorization");

        if (authHeader == null || !authHeader.startsWith("Bearer ")) {
            response.setStatus(401);
            response.setContentType("application/json;charset=utf-8");
            response.getWriter().write("{\"code\":401,\"msg\":\"未登录\",\"data\":null}");
            return;
        }

        String token = authHeader.substring(7);  // 去掉 "Bearer " 前缀

        try {
            // 解析 Token，取出 userId
            Long userId = jwtUtil.getUserId(token);
            // 存入 request，后续 Controller 通过 request.getAttribute("userId") 获取
            request.setAttribute("userId", userId);
            // 放行
            filterChain.doFilter(request, response);
        } catch (Exception e) {
            response.setStatus(401);
            response.setContentType("application/json;charset=utf-8");
            response.getWriter().write("{\"code\":401,\"msg\":\"Token无效或已过期\",\"data\":null}");
        }
    }
}
```

### Step 2.5 注册 Filter

**文件：** `src/main/java/com/mj/aiknowledgebase/config/WebConfig.java`

**知识点：**
- `FilterRegistrationBean` — 把自定义 Filter 注册到 Spring 容器
- `setOrder(1)` — 执行顺序，数字越小越先执行
- `addUrlPatterns("/api/*")` — 只拦截 /api/ 开头的请求

```java
package com.mj.aiknowledgebase.config;

import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.boot.web.servlet.FilterRegistrationBean;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.web.servlet.config.annotation.CorsRegistry;
import org.springframework.web.servlet.config.annotation.WebMvcConfigurer;

@Configuration
public class WebConfig implements WebMvcConfigurer {

    @Autowired
    private JwtFilter jwtFilter;

    // 注册 JWT Filter
    @Bean
    public FilterRegistrationBean<JwtFilter> jwtFilterRegistration() {
        FilterRegistrationBean<JwtFilter> registration = new FilterRegistrationBean<>();
        registration.setFilter(jwtFilter);
        registration.addUrlPatterns("/api/*");
        registration.setOrder(1);
        return registration;
    }

    // CORS 跨域配置
    @Override
    public void addCorsMappings(CorsRegistry registry) {
        registry.addMapping("/**")
                .allowedOriginPatterns("*")
                .allowedMethods("GET", "POST", "PUT", "DELETE", "OPTIONS")
                .allowedHeaders("*")
                .allowCredentials(true);
    }
}
```

### Step 2.6 创建 UserService

**文件：** `src/main/java/com/mj/aiknowledgebase/service/UserService.java`

**知识点：**
- `Service` 接口 + `Impl` 实现类 — 分离定义和实现，便于后续扩展和测试
- `@Service` — 声明为 Spring Bean
- `BCryptPasswordEncoder` — Spring Security 提供的密码加密工具（Spring Boot 自带）
  - `.encode("123456")` → `$2a$10$xxxxxxxxxxxx`（每次结果不同，因为加了随机盐）
  - `.matches("123456", hash)` → true/false（验证密码）

```java
package com.mj.aiknowledgebase.service;

import com.baomidou.mybatisplus.core.conditions.query.LambdaQueryWrapper;
import com.mj.aiknowledgebase.entity.User;
import com.mj.aiknowledgebase.mapper.UserMapper;
import com.mj.aiknowledgebase.util.JwtUtil;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.security.crypto.bcrypt.BCryptPasswordEncoder;
import org.springframework.stereotype.Service;

@Service
public class UserService {

    @Autowired
    private UserMapper userMapper;

    @Autowired
    private JwtUtil jwtUtil;

    private final BCryptPasswordEncoder passwordEncoder = new BCryptPasswordEncoder();

    // 注册
    public User register(String username, String password) {
        // 检查用户名是否已存在
        LambdaQueryWrapper<User> wrapper = new LambdaQueryWrapper<>();
        wrapper.eq(User::getUsername, username);
        if (userMapper.selectOne(wrapper) != null) {
            throw new RuntimeException("用户名已存在");
        }

        // 创建用户
        User user = new User();
        user.setUsername(username);
        user.setPassword(passwordEncoder.encode(password));  // 加密存储
        userMapper.insert(user);

        user.setPassword(null);  // 返回时隐藏密码
        return user;
    }

    // 登录
    public String login(String username, String password) {
        // 查询用户
        LambdaQueryWrapper<User> wrapper = new LambdaQueryWrapper<>();
        wrapper.eq(User::getUsername, username);
        User user = userMapper.selectOne(wrapper);

        if (user == null) {
            throw new RuntimeException("用户名或密码错误");
        }

        // 验证密码
        if (!passwordEncoder.matches(password, user.getPassword())) {
            throw new RuntimeException("用户名或密码错误");
        }

        // 生成 JWT Token
        return jwtUtil.generateToken(user.getId(), user.getUsername());
    }
}
```

### Step 2.7 创建 AuthController

**文件：** `src/main/java/com/mj/aiknowledgebase/controller/AuthController.java`

**知识点：**
- `@RestController` — 等于 @Controller + @ResponseBody，返回值自动转 JSON
- `@PostMapping("/api/register")` — 映射 POST 请求
- `@RequestBody` — 把请求体的 JSON 自动转成 Java 对象

```java
package com.mj.aiknowledgebase.controller;

import com.mj.aiknowledgebase.common.R;
import com.mj.aiknowledgebase.entity.User;
import com.mj.aiknowledgebase.service.UserService;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.web.bind.annotation.*;

import java.util.Map;

@RestController
public class AuthController {

    @Autowired
    private UserService userService;

    // 注册
    @PostMapping("/api/register")
    public R<?> register(@RequestBody Map<String, String> body) {
        String username = body.get("username");
        String password = body.get("password");

        if (username == null || password == null) {
            return R.error("用户名和密码不能为空");
        }

        User user = userService.register(username, password);
        return R.ok("注册成功", user);
    }

    // 登录
    @PostMapping("/api/login")
    public R<?> login(@RequestBody Map<String, String> body) {
        String username = body.get("username");
        String password = body.get("password");

        if (username == null || password == null) {
            return R.error("用户名和密码不能为空");
        }

        String token = userService.login(username, password);
        return R.ok("登录成功", Map.of("token", token));
    }
}
```

```java
import com.baomidou.mybatisplus.core.conditions.query.LambdaQueryWrapper;

import com.mj.aiknowledgebase.entity.Knowledge;

import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Service;

import java.util.List;

@Service
public class KnowledgeService {

    @Autowired
    private KnowledgeMapper knowledgeMapper;

    // 创建知识库
    public Knowledge create(Long userId, String name, String description) {
        Knowledge knowledge = new Knowledge();
        knowledge.setUserId(userId);
        knowledge.setName(name);
        knowledge.setDescription(description);
        knowledgeMapper.insert(knowledge);
        return knowledge;
    }

    // 查询当前用户的知识库列表
    public List<Knowledge> listByUserId(Long userId) {
        LambdaQueryWrapper<Knowledge> wrapper = new LambdaQueryWrapper<>();
        wrapper.eq(Knowledge::getUserId, userId)
               .orderByDesc(Knowledge::getCreateTime);
        return knowledgeMapper.selectList(wrapper);
    }

    // 分页查询（示例，当前接口先不用，留着练手）
    public Page<Knowledge> pageByUserId(Long userId, int pageNum, int pageSize) {
        Page<Knowledge> page = new Page<>(pageNum, pageSize);
        LambdaQueryWrapper<Knowledge> wrapper = new LambdaQueryWrapper<>();
        wrapper.eq(Knowledge::getUserId, userId)
               .orderByDesc(Knowledge::getCreateTime);
        return knowledgeMapper.selectPage(page, wrapper);
    }

    // 查询单个（检查是否属于当前用户）
    public Knowledge getByIdAndUserId(Long id, Long userId) {
        LambdaQueryWrapper<Knowledge> wrapper = new LambdaQueryWrapper<>();
        wrapper.eq(Knowledge::getId, id)
               .eq(Knowledge::getUserId, userId);
        return knowledgeMapper.selectOne(wrapper);
    }

    // 删除知识库
    public void delete(Long id, Long userId) {
        // 先检查是否存在且属于当前用户
        Knowledge knowledge = getByIdAndUserId(id, userId);
        if (knowledge == null) {
            throw new RuntimeException("知识库不存在或无权删除");
        }
        knowledgeMapper.deleteById(id);
    }
}
```

### Step 3.2 创建 KnowledgeController

**文件：** `src/main/java/com/mj/aiknowledgebase/controller/KnowledgeController.java`

**知识点：**
- `@GetMapping` / `@DeleteMapping` — 不同 HTTP 方法映射
- `@PathVariable` — 从 URL 路径中取参数，如 `/api/knowledge/{id}` 中的 id
- `request.getAttribute("userId")` — 从 Filter 中存入的当前用户 ID

```java
package com.mj.aiknowledgebase.controller;

import com.mj.aiknowledgebase.common.R;
import com.mj.aiknowledgebase.entity.Knowledge;
import com.mj.aiknowledgebase.service.KnowledgeService;
import jakarta.servlet.http.HttpServletRequest;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.web.bind.annotation.*;

import java.util.Map;

@RestController
public class KnowledgeController {

    @Autowired
    private KnowledgeService knowledgeService;

    // 从 request 中获取当前用户 ID（Filter 存入的）
    private Long getCurrentUserId(HttpServletRequest request) {
        return (Long) request.getAttribute("userId");
    }

    // 创建知识库
    @PostMapping("/api/knowledge")
    public R<?> create(@RequestBody Map<String, String> body, HttpServletRequest request) {
        Long userId = getCurrentUserId(request);
        String name = body.get("name");
        String description = body.get("description");

        if (name == null || name.isBlank()) {
            return R.error("知识库名称不能为空");
        }

        Knowledge knowledge = knowledgeService.create(userId, name, description);
        return R.ok("创建成功", knowledge);
    }

    // 知识库列表
    @GetMapping("/api/knowledge")
    public R<?> list(HttpServletRequest request) {
        Long userId = getCurrentUserId(request);
        return R.ok(knowledgeService.listByUserId(userId));
    }

    // 删除知识库
    @DeleteMapping("/api/knowledge/{id}")
    public R<?> delete(@PathVariable Long id, HttpServletRequest request) {
        Long userId = getCurrentUserId(request);
        knowledgeService.delete(id, userId);
        return R.ok("删除成功", null);
    }
}
```

### Step 4.1 创建 DocumentService

**文件：** `src/main/java/com/mj/aiknowledgebase/service/DocumentService.java`

**知识点：**
- `MultipartFile` — Spring 接收上传文件的对象
- `file.getOriginalFilename()` — 获取原始文件名
- `file.transferTo(dest)` — 把文件保存到磁盘
- `UUID.randomUUID()` — 生成随机文件名，避免重名覆盖
- `substring(lastIndexOf("."))` — 取文件后缀名

```java
package com.mj.aiknowledgebase.service;

import com.baomidou.mybatisplus.core.conditions.query.LambdaQueryWrapper;
import com.mj.aiknowledgebase.entity.Document;
import com.mj.aiknowledgebase.mapper.DocumentMapper;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.beans.factory.annotation.Value;
import org.springframework.stereotype.Service;
import org.springframework.web.multipart.MultipartFile;

import java.io.File;
import java.io.IOException;
import java.util.List;
import java.util.UUID;

@Service
public class DocumentService {

    @Autowired
    private DocumentMapper documentMapper;

    @Value("${file.upload-dir}")
    private String uploadDir;

    // 上传文档
    public Document upload(Long knowledgeId, MultipartFile file) throws IOException {
        // 1. 获取文件信息
        String originalName = file.getOriginalOriginalFilename();
        String fileType = originalName.substring(originalName.lastIndexOf(".") + 1);
        long fileSize = file.getSize();

        // 2. 生成存储文件名（UUID 避免重名）
        String storedName = UUID.randomUUID().toString() + "." + fileType;

        // 3. 确保上传目录存在
        File dir = new File(uploadDir);
        if (!dir.exists()) {
            dir.mkdirs();
        }

        // 4. 保存文件到磁盘
        File dest = new File(dir, storedName);
        file.transferTo(dest);

        // 5. 保存记录到数据库
        Document document = new Document();
        document.setKnowledgeId(knowledgeId);
        document.setFileName(originalName);
        document.setFileType(fileType);
        document.setFileSize(fileSize);
        document.setFilePath(dest.getAbsolutePath());
        document.setStatus(0);  // 待处理
        documentMapper.insert(document);

        return document;
    }

    // 查询知识库下的文档列表
    public List<Document> listByKnowledgeId(Long knowledgeId) {
        LambdaQueryWrapper<Document> wrapper = new LambdaQueryWrapper<>();
        wrapper.eq(Document::getKnowledgeId, knowledgeId)
               .orderByDesc(Document::getCreateTime);
        return documentMapper.selectList(wrapper);
    }

    // 查询文档详情
    public Document getById(Long id) {
        return documentMapper.selectById(id);
    }
}
```

```java
package com.mj.aiknowledgebase.controller;

import com.mj.aiknowledgebase.common.R;
import com.mj.aiknowledgebase.entity.Document;

import org.springframework.beans.factory.annotation.Autowired;

public class DocumentController {

    public R<?> detail(@PathVariable Long id) {
        Document document = documentService.getById(id);
        if (document == null) {
            return R.error("文档不存在");

    }
```

### Step 4.3 Redis 练手（可选）

**文件：** `src/main/java/com/mj/aiknowledgebase/config/RedisConfig.java`

**知识点：**
- `RedisTemplate` — Spring 操作 Redis 的核心类
- `StringRedisTemplate` — 简化版，key 和 value 都是 String
- `opsForValue().set(key, value, timeout)` — 存
- `opsForValue().get(key)` — 取
- `delete(key)` — 删

```java
package com.mj.aiknowledgebase.config;

import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.data.redis.connection.RedisConnectionFactory;
import org.springframework.data.redis.core.StringRedisTemplate;

@Configuration
public class RedisConfig {

    @Bean
    public StringRedisTemplate stringRedisTemplate(RedisConnectionFactory factory) {
        return new StringRedisTemplate(factory);
    }
}
```

**在 UserService.login() 中加入 Redis 用法（理解即可，不强制）：**

```java
// 在 UserService 中注入
@Autowired
private StringRedisTemplate redisTemplate;

// login() 方法中，生成 token 后加上：
String redisKey = "token:" + user.getId();
redisTemplate.opsForValue().set(redisKey, token, 24, TimeUnit.HOURS);
```

```java
// 在 JwtFilter 的验证逻辑中，可以加上 Redis 校验：
String redisKey = "token:" + userId;
String cachedToken = redisTemplate.opsForValue().get(redisKey);
if (cachedToken == null || !cachedToken.equals(token)) {
    // Token 不在 Redis 中，已失效
}
```

**这样理解 Redis 的作用：** 登录时把 Token 存 Redis，验证时从 Redis 查。退出时删 Redis 的 key，Token 立刻失效。
